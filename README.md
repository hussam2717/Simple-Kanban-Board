## Simple-Kanban-Board
A clean and minimal Kanban Board built using HTML, CSS, and JavaScript. This project helps you manage tasks across different stages like To Do, In Progress, and Done.

## Live Demo - https://hussam2717.github.io/Simple-Kanban-Board/

## 🚀 Features
📌 Drag and drop tasks between columns
🧩 Simple and clean UI design
⚡ Lightweight and fast (no frameworks)
📱 Responsive layout

## 🛠️ Tech Stack
HTML5
CSS3
JavaScript

## 📂 Project Structure
├── index.html
├── style.css
├── script.js

🎯 How It Works
Tasks are displayed in 3 columns:
To Do
In Progress
Done
You can drag tasks from one column to another to update their status.
